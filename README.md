# ReadyRoom Project

Fullstack Hotel Booking Platform developed for ReadyRoom.

---

## About This Project

ReadyRoom is a hotel booking platform designed to support modern hotel operations including **transit bookings (3-hour system)**, room management, and centralized admin control.

This project is developed as a **Full Stack Web & Mobile System**, starting from the initial system architecture, database design, backend API development, and frontend admin dashboard.

The platform is designed to support future integrations such as **mobile applications, hotel partner systems, and payment gateway services**.

## System Modules

### Hotel Management
Admin can:

- Add new hotel branches
- Upload hotel images
- Manage hotel locations
- Configure hotel facilities

### Room Management

Admin can:

- Add room types
- Set room pricing
- Manage room availability

### Booking System

Customers can:

- Select city
- Choose hotel
- Select room
- Choose duration
- Create booking
